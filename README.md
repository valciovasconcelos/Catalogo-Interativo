# Catálogo Interativo de Produtos

Este é um projeto de catálogo de produtos desenvolvido com **Next.js** e **React**. O objetivo é demonstrar o uso de componentes reutilizáveis, roteamento dinâmico e gerenciamento de estado em uma aplicação moderna.

## Funcionalidades

- **Navegação Responsiva**: Menu acessível em dispositivos móveis e desktops.
- **Listagem de Produtos**: Visualização de diversos itens de tecnologia.
- **Busca em Tempo Real**: Filtro dinâmico na página de produtos.
- **Páginas de Detalhes**: Cada produto possui sua própria página com informações detalhadas.
- **Interatividade**: Sistema de "Curtidas" com contador individual.

## Estrutura do Projeto

- `/app`: Contém as rotas e páginas da aplicação.
- `/components`: Componentes reutilizáveis (Header, Footer, Card, etc).
- `/data`: Dados mockados dos produtos.
- `/public`: Arquivos estáticos e imagens.

## Como Executar

Certifique-se de ter o Node.js instalado.

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
